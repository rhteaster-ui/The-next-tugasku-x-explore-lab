==================================================
GUEST MODE
==================================================

Jangan matikan guest/local mode.

TugasKu harus tetap bisa digunakan tanpa login untuk basic usage.

Buat mode:

1. Guest Mode
   - data tersimpan lokal di IndexedDB/localStorage
   - tidak sync antar device
   - tetap bisa pakai basic task/notes/focus
   - tampilkan prompt halus untuk login

2. Signed-in Mode
   - data bisa tersimpan ke cloud
   - sync antar device
   - bisa backup/restore
   - bisa connect Google Calendar
   - AI history bisa disimpan lebih aman

Jangan paksa login terlalu agresif.
Buat UX halus.

Contoh CTA:
- “Lanjut tanpa login”
- “Masuk untuk sinkronisasi”
- “Backup data lokal ke akun”

==================================================
GOOGLE CALENDAR INTEGRATION
==================================================

Siapkan pondasi untuk Google Calendar integration sebagai fitur opt-in.

Jangan langsung paksa permission Calendar saat login utama.

Flow yang diinginkan:
1. User login ke TugasKu
2. Di settings atau calendar page ada tombol:
   “Connect Google Calendar”
3. User memberi izin Calendar
4. TugasKu bisa membuat event dari task/deadline
5. Reminder berjalan lewat Google Calendar app di HP

Calendar permission harus terpisah dari login biasa agar UX lebih bersih.

Fitur minimal:
- connect Google Calendar
- disconnect Google Calendar
- create calendar event from task
- update event if task due date changes
- remove/unlink event if task deleted or completed
- choose reminder default:
  - 10 minutes before
  - 1 hour before
  - 1 day before
- store google_calendar_event_id in task

Route/API yang disarankan:
- `/api/integrations/google-calendar/connect`
- `/api/integrations/google-calendar/callback`
- `/api/integrations/google-calendar/disconnect`
- `/api/integrations/google-calendar/create-event`
- `/api/integrations/google-calendar/update-event`
- `/api/integrations/google-calendar/delete-event`

UI:
- di `/settings/integrations`
- di `/tugasku/calendar`
- di task detail modal

Copy:
- “Gunakan Google Calendar untuk reminder yang tetap aktif meskipun web app tidak sedang dibuka.”
- “TugasKu akan membuat event kalender dari deadline tugasmu.”
