PENEGASAN, JANGAN GUNAKAN EMOJI SEBAGAI ICON PADA ICON2 WEB NYA, GUNAKAN GAMBAR YG SELAIN DARI EMOJI.
=====
Buatkan sebuah website portfolio developer yang modern, clean, dan premium dengan aesthetic dingin, tenang, dan minimal.

Arah desain:
- Gunakan warna dasar monochrome (hitam pekat, abu gelap, putih lembut)
- Gunakan satu warna aksen: biru dingin tidak wajib karena terlalu umum, ganti warna cool lainnya
- Mood keseluruhan: dingin, tenang, elegan, sedikit futuristik tapi tidak berlebihan
- Tampilan harus terasa mahal, dan percaya diri

Background:
- Gunakan background gradient berlapis (bukan warna flat)
- Kombinasikan 2–3 warna dingin: biru (boleh bukan wajib) gelap, putih lembut, abu tua
- Tambahkan radial gradient atau efek cahaya blur untuk memberi kedalaman (seolah 3D)
- Opsional: tambahkan noise/grain halus
- Background harus terasa hidup tapi tetap lembut dan tidak mengganggu

Elemen UI:
- Gunakan gaya glassmorphism (efek kaca buram)
  - card semi-transparan
  - efek blur di belakang (backdrop blur)
  - border tipis dengan opacity rendah
- Sudut elemen rounded (sedang)
- Shadow lembut (hindari shadow keras)
- Divider garis tipis dan halus

Typography:
- Gunakan font modern dan clean (seperti Inter)
- Hierarki teks jelas
- Gunakan teks minimal tapi tetap jelas
- Beri sedikit jarak antar huruf pada heading

Layout:
- Banyak whitespace (jangan padat)
- Gunakan grid layout
- Struktur section:
  1. Hero (nama + tagline singkat)
  2. About (deskripsi singkat dalam glass card)
  3. Projects (grid card dengan efek hover)
  4. Tech stack (ikon minimal)
  5. Contact (sederhana dan clean)

Animasi:
- Gunakan animasi halus dan pelan
- Fade + sedikit gerakan ke atas
- Efek blur menjadi jelas
- Hindari animasi cepat atau memantul

Interaksi:
- Hover dengan glow biru tipis
- Scale sangat kecil (subtle)
- Opsional: efek cursor glow

Teknis:
- Gunakan stack modern (TypeScript + Tailwind CSS)
- Harus responsive (mobile tetap ringan dengan efek disederhanakan)
- Optimalkan performa (hindari 3D berat di mobile)

Catatan penting:
- Hindari tampilan ramai
- Hindari banyak warna
- Hindari efek berlebihan
- Fokus pada kesan tenang, premium, dan clean
Buat background dengan efek gradient berlapis yang terasa seperti memiliki kedalaman (semi 3D).

Detail:
- Gunakan beberapa layer gradient (bukan satu gradient saja)
- Kombinasikan hitam pekat dengan biru gelap dan ungu lembut
- Tambahkan radial gradient untuk efek cahaya
- Gunakan blur pada beberapa area agar terlihat seperti glow
- Bisa tambahkan bentuk abstrak (blob) yang sangat halus
- Animasi sangat pelan (opsional)

Karakter background:
- Dalam (deep)
- Atmosferik
- Dingin
- Modern
- Premium

Hindari:
- Warna terang mencolok
- Kontras terlalu tinggi
- Animasi cepat
